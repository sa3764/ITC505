Addendum: Stylistic Choices

Font Choices

I used Georgia for headings because it conveys a formal, academic tone. The body text uses Arial, a modern sans-serif font that is easy to read on screens.

Heading and Text Sizes

The title (h1) is 40px for prominence. Section headings (h2) are 28px, and subheadings (h3) are 22px. The paragraph text is set to 18px for comfortable reading distance and flow.

Color Choices

I chose a pink and navy palette for a soft yet elegant appearance. The background is light pink for warmth, and text uses darker shades for contrast.

Margins and Padding

The document includes generous padding (25–40px) to maintain white space and focus attention on content. Each section has soft borders and shadows to define structure and improve readability.

HTML Abuse

In Lab 2, I used an <h2> tag for the main title purely for visual reasons. This has now been corrected to <h1> to preserve semantic structure. All visual adjustments are now handled with CSS, not HTML.
